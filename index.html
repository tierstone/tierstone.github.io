Who We Are

Hey there. Tierstone Foundation is basically just a group of us who love building open-source software. We started out as TuxForge, but eventually decided we wanted to stop treating it like a chaotic hobby and actually build things the right way—with solid architecture and code we can be proud of.

We think open source should be more than just "hey, it compiles." We want to write code that's clean, robust, and built to survive. No more duct-tape solutions—we're here to build tools you can actually rely on without losing your mind.

The OS

A 64-bit OS written from scratch in C. It runs exclusively on QEMU x86_64. We're keeping things incredibly lean, strictly structured, and focused purely on modern architecture.

GPLv3 C99/C11 (Strict)

Tech Stack

CompilerClang/LLVM
LibCmusl
TargetQEMU x86_64 (q35, OVMF)
DriversVirtIO only
(Net, Block, Console, RNG)

AI POLICY: HOSTILE

Local OSS LLMs may be used for syntax lookup or boilerplate scaffolding only. Zero tolerance for committing unreviewed AI output.

"If your code smells like it was generated by a probabilistic token predictor without human understanding, it will be rejected. We build systems, we don't prompt them."

Build & Run

Prerequisites: Clang/LLVM (`clang`, `lld-link`), Python 3, QEMU, OVMF (`edk2-ovmf`).

# Build the EFI binary

python3 build.py all

# Launch in QEMU

python3 boot.py
