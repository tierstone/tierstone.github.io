Organization

A software organization,
not a mood board.

Tierstone is an engineering-focused open-source organization. It grew out of the Platinum Foundation and was restructured around a simpler standard: build fewer things, build them carefully, and leave them in a state other people can actually maintain.

The emphasis is on stable architecture, readable code, and systems that can be reasoned about without mythology. That means less posturing, less ornamental design language, and more straightforward engineering.

Principles

Correctness first

Fewer moving parts, tighter assumptions, less magic.

Maintainability over novelty

Code should still make sense six months later, not just look clever today.

Long-term usefulness

Projects should remain solid foundations, not one-week experiments with branding.

Project spotlight

Platinum (/p/OS)

Repository

Overview

Platinum is a 64-bit operating system written from scratch in strict C for QEMU x86_64. The work is intentionally narrow in scope: clear bring-up, disciplined systems code, and a kernel that grows from firm ground instead of vague ambition.

GPLv3 C99/C11 x86_64 QEMU UEFI

AI policy

Human-reviewed code only

AI tools may be used for reference, exploration, and boilerplate generation. They are tools, not authors.

Automated or agentic workflows that generate and publish unreviewed code are not acceptable. If the contributor cannot explain the code, it does not belong in the tree.
